Account summary list

This component can be used to show an overview of all connected accounts

Example code


    <certua-ob-account-summary-list 
    [daasUrl]="daasUrl"
      [contextData]="contextData">
    </certua-ob-account-summary-list>
Property Name Mandatory Description
sortOptions No This is JSON which allows you to set the sortFieldName and sortOrder. Currently limited to sortFieldName must be 'balance' and sortOrder 'asc' or 'desc'. If this is not passed then they are ordered by Provider name
{{ sortOptions | json }}
limitTo No This is an integer that controls how many accounts are shown.

You do not have a context token
