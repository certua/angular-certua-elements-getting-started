Account summary list

This component can be used to show an overview of all connected accounts

Example code


    <certua-ob-account-summary-list
      [daasUrl]="daasUrl"
      [contextData]="contextData">
    </certua-ob-account-summary-list>

Component specific inputs

Property Name Mandatory Region Description
sortOptions No All This is JSON which allows you to set the sortFieldName and sortOrder. Currently limited to sortFieldName must be 'balance' and sortOrder 'asc' or 'desc'. If this is not passed then they are ordered by Provider name
{{ sortOptions | json }}

Common inputs

Event bus messages

This component emits the total value of the account balances grouped by currency with the event type of account-summary-balances 


   {{this.accountSummaryEvent |json}}
