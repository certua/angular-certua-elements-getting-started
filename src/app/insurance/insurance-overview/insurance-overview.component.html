Embedding our components

No matter what coding language your site is written in you can easily embed our components with minimal html and javascript. Using our example codebase and component explanations embedding one or more of our components into your site should be super simple for any developer.

How our components work together

We have multiple components that work together to build up the full insurance purchasing and servicing experience. You can choose which components you wish to use.

  • Quote and buy - this is the main component which houses the underwriting and purchasing journey
  • Claims- this displays a short snippet of text (provided by you) about how a customer can make a claim
  • First Notification of Loss (FNOL) - this displays a form a customer can fill in to make a claim
  • Quick Quote - this allows you to give out indicative quotes based on a subset of questions
  • Login - if using Certua's security this simply displays a login button and logout button based on your authentication status
  • View Policy v2- This is one component that allows a customer to view their active policy as well as their previous quotes/policies

Legacy components

We recommend using our newer components but these are available if required

  • View Policy - v1 - This simply displays a single policy based on the policy Id
  • Quotes list - This displays all quotes for a customer
  • Policies list - This displays all policies for a customer
