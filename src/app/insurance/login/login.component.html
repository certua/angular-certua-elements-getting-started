Login

This component allows you to login to the Certua Insurance platform and then access the other components including View Policy. You will need to pass in what the component is to display for the login and logout buttons.

Example code

      
    <certua-insurance-login
    [referrerSiteCode]="config.referrerId"
    [coversUrl]="'http://localhost:4200/angular/components/view-policy2'"
    [loginUrl]="'http://localhost:4200/angular/components/view-policy2'"
    [logoutUrl]="'http://localhost:4200/angular/components/login'">

    <button class="btn btn-primary" slot="login">Login</button>
    <button class="btn btn-secondary btn-circle" slot="logout"><i class="fa fa-user"></button>

    </certua-insurance-login >

Input Elements

Component specific inputs

Property Name Mandatory Description
loginUrl Yes This is the url that the user will be redirected to after a successful login. This page would usually host the View Policy component.
logoutUrl Yes This is the url that the user will be redirected to after a successful logout.
coversUrl Yes This is the url that the user will be taken to when they choose the My Covers option from the logged in menu. This page would usually host the View Policy component.
