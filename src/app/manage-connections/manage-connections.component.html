Manage connections

This component can be used to view connected accounts, refresh consent or disconnect

Example code


    <certua-ob-manage-connections
      [contentOverrides]="contentOverrides"
      [contextData]="contextData"
      [showTitle]="false"
      [showAddButton]="false">
    </certua-ob-manage-connections>
Property Name Mandatory Description
contextData Yes This is a JSON string which contains your context token and user reference.
{{ '{"contextToken":"FF5D16AAE1ED74E4C8F0E8B6D9E2EB06","ownerId":"1","dateCreated":"2023-03-20T11:17:24.121Z"}' }}
contentOverrides No This is a JSON string which contains any text overrides that you want to pass to the information screens/modals
{{ contentOverrides | json }}
showTitle No This will show or hide the web components title (Open Banking Connected Accounts)
showAddButton No This controls if the Add Account button is shown above the currently connected accounts.
daasURL No You can provide a full url to where you have deployed the daas elements library. This can be used when it is not possible to use the default cdn url hosted by Certua.

You do not have a context token
