Manage connections

This component can be used to view connected accounts, refresh consent or disconnect

Example code


    <certua-ob-manage-connections
      [contentOverrides]="contentOverrides"
      [contextData]="contextData"
      [showTitle]="false"
      [daasUrl]="daasUrl"
      [redirectionConfig]="redirectionConfig"
      [showAddButton]="false"
      [editAccountName]="false">

    </certua-ob-manage-connections>

Component specific inputs

Property Name Mandatory Region Description
contentOverrides No UK only This is a JSON string which contains any text overrides that you want to pass to the information screens/modals
{{ contentOverrides | json }}
showTitle No All This will show or hide the web components title (Open Banking Connected Accounts)
showAddButton No All This controls if the Add Account button is shown above the currently connected accounts.
editAccountName No All This controls if the Rename Account button is shown when managing connected accounts.
redirectionConfig Yes All This is a JSON string which contains the success or failure redirection Urls, which are used after the connection to the bank. Popup controls whether or not the bank url is shown in the same tab or in a new tab, this defaults to false if not present
{{ redirectionConfig | json }}
